# TechBlo 博客平台项目说明

## 项目概述

本项目是一个基于 Next.js 和 Node.js 的现代化博客平台，实现了登录页面和博客主页面，支持 GitHub OAuth 登录和用户名/密码登录。

## 项目结构

```
boke/
├── boke_web/              # 前端项目 (Next.js)
│   ├── app/              # Next.js App Router
│   │   ├── page.tsx      # 登录页面路由
│   │   ├── blog/         # 博客主页面路由
│   │   └── auth/         # OAuth 回调路由
│   ├── components/       # React 组件
│   │   ├── LoginPage.tsx # 登录页面组件
│   │   └── BlogPage.tsx  # 博客页面组件
│   └── ...
├── boke_server/          # 后端项目 (Node.js)
│   ├── server.js         # 主服务器文件
│   ├── package.json      # 依赖配置
│   └── ...
└── README.md             # 项目说明
```

## 技术栈

### 前端
- **Next.js 14** - React 框架（App Router）
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **Axios** - HTTP 客户端

### 后端
- **Node.js** - 运行环境
- **Express** - Web 框架
- **Passport.js** - 认证中间件
- **passport-github2** - GitHub OAuth 策略
- **passport-jwt** - JWT 认证策略
- **bcryptjs** - 密码加密
- **jsonwebtoken** - JWT 生成和验证

## 启动步骤

详细步骤请参考 `QUICK_START.md`

1. 安装依赖（前后端）
2. 配置 GitHub OAuth App
3. 配置后端环境变量
4. 启动后端服务（端口 3001）
5. 启动前端服务（端口 3000）
6. 访问 http://localhost:3000

## 注意事项

⚠️ **当前实现**
- 使用内存存储用户数据（重启后数据会丢失）
- 使用模拟博客数据
- 开发环境配置（未启用 HTTPS）

⚠️ **生产环境建议**
- 使用真实数据库（MongoDB、PostgreSQL 等）
- 配置 HTTPS
- 使用环境变量管理敏感信息
- 添加输入验证和错误处理
- 实现博客文章的 CRUD 操作
- 添加用户权限管理
- 实现博客搜索和筛选功能

## API 接口

### 认证接口

- `GET /api/auth/github` - GitHub OAuth 登录
- `GET /api/auth/github/callback` - GitHub OAuth 回调
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/login` - 用户登录
- `GET /api/auth/me` - 获取当前用户信息（需要认证）

### 博客接口

- `GET /api/posts` - 获取博客列表（需要认证）
- `GET /api/posts/:id` - 获取单个博客文章（需要认证）

## 开发建议

1. **数据库集成**
   - 使用 MongoDB 或 PostgreSQL 存储用户和博客数据
   - 使用 Mongoose 或 Sequelize 作为 ORM

2. **功能扩展**
   - 博客文章 CRUD 操作
   - 用户头像上传

   - 评论功能
   - 标签系统

3. **用户体验**
   - 加载状态提示
   - 错误提示优化
   - 表单验证
   - 响应式设计优化

4. **安全加固**
   - 输入验证和清理
   - 速率限制
