## 💡 提示和技巧

### 开发技巧
1. **保存自动刷新** - Vite HMR 无需手动刷新
2. **类型提示** - VS Code 提供完整的 TypeScript 提示
3. **组件化开发** - 每个组件独立文件和样式
4. **快捷键** - 使用 Ctrl+S 保存，F12 打开开发者工具

### 常见问题
1. **端口被占用？** - 修改 vite.config.ts 中的端口
2. **类型报错？** - 检查 tsconfig.json 配置
3. **样式不生效？** - 确保正确导入 CSS 文件
4. **热更新失败？** - 重启开发服务器
